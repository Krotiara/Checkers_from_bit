ОПИСАНИЕ
Данный скрипт является реализацией "PEP8 валидатора".

ТРЕБОАНИЯ
1. Python версии не ниже 3.4

СОСТАВ
1. Консольная версия: pep8_console.py
2. Тесты: pep8_tests.py
3. Файл с кодом: pep8.py

КОНСОЛЬНАЯ ВЕРСИЯ
Справка по запуску: pep8_console.py --help
Пример запуска: pep8_console.py example1.txt
На выходе вы увидите возможно допущенные вами ошибки в вашем коде.

ПОДРОБНОСТИ РЕАЛИЗАЦИИ
В pep8.py есть класс PepValidator, в котором содержится алгоритм 
"PEP8 валидатора".

И пусть удача всегда будет с вами...
